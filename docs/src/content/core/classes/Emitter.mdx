# Emitter

The `Emitter` class is an `abstract class` that provides the ability to register event listeners, dispatch events.


## Methods

The `Emitter` class provides the following methods:

| Method               | Description                                |
|----------------------|--------------------------------------------|
| `on`                 | Register event listeners                   |
| `once`               | Register one-time event listeners          |
| `dispatch`           | Dispatch events                            |
| `removeAllListeners` | Remove all listeners                       |

## Example

To use Emitter, extend it with a specific event map.

```ts
import { Emitter } from '@fewings/core/classes';

type MyEvents = {
  click: (x: number, y: number) => void;
  message: (text: string) => void;
};

class MyEmitter extends Emitter<MyEvents> {
    constructor() {
        super();
    }
}

const emitter = new MyEmitter();

// Register event listeners
const offClick = emitter.on('click', (x, y) => {
  console.log('click', x, y);
});

const offMessage = emitter.on('message', (text) => {
  console.log('message', text);
});

// Dispatch events
emitter.dispatch('click', 10, 20);
emitter.dispatch('message', 'Hello, World!');

// Remove the event listeners
offClick();
offMessage();

// removeAllListeners
emitter.removeAllListeners();
```
